# QuickFix:  Home Repair & Services Platform

QuickFix is a MERN stack web application that connects users with trusted service providers for quick and reliable home repairs. Whether it’s plumbing, electrical work, carpentry, or general maintenance, QuickFix makes it easy to book services, manage requests, and track task status — all in one place.

 #Features
# User Side

Create an account & login securely

Browse available repair/maintenance services

Book a service with preferred date & time

Track booking status (Pending → Approved → Completed)

Manage profile & view booking history

# Service Provider Side

Separate provider login

Manage received job requests

Accept / reject tasks

Update job status

View previous service records
